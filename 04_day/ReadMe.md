# פעולות עריכה, מיון וסינון

במסמך זה נדבר על פעולות עריכה, מיון וסינון שניתן לבצע על נתונים.

## עריכה

פעולת העריכה מאפשרת למשתמש לשנות פריטי מידע קיימים. לדוגמה, ניתן לשנות ערכים של רשומות קיימות או להוסיף מידע חדש.

## מיון

מיון הוא תהליך של ארגון רשימה בהתאם לקריטריונים מסוימים. לדוגמה, ניתן למיין רשימה של מספרים מהקטן לגדול, או לסדר מערך של אובייקטים לפי תכונה מסוימת.

## סינון

פעולת הסינון משמעה לקבוע אילו פריטים ייכללו מתוך רשימה. לדוגמה, ניתן לסנן רשימה כך שתוצג רק רשומות שמקיימות תנאים מסוימים.

## דוגמה לשימוש ב-JavaScript

```javascript
let array = [5, 3, 8, 1];

// מיון
let sortedArray = array.sort((a, b) => a - b);

// סינון
let filteredArray = sortedArray.filter(num => num > 3);
```
